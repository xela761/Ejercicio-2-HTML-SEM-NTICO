¿Qué es el HTML semántico y por qué es importante?

Son las etiquetas propio de HTML que envian o transmiten el significado de su contenido, es importante ya que da un orden especifico para poder realizar una pagina web mas ordenada y con una interfaz mas amigable al consumidor de dicha pagina que se va a realizar.

Cómo funciona el HTML semántico

Dando etiquetas a la cada parte de la estructura es mas sencillo trabajar con dicha programación.

"Las etiquetas HTML semánticas son etiquetas que definen el significado del contenido que engloban" source:https://es.semrush.com/blog/html-semantico/"

Ejemplos de HTML con etiquetas semánticas

Etiquetas como "header", "article", "footer" son HTML semánticas.

Ejemplos de HTML sin etiquetas semánticas

Unos ejemplos de etiquetas HTML no semánticas son "div" y "span".

Algunas etiquetas de HTML5 semántico

  • nav
  • main
  • section
  • article
  • footer
